Field Data Submitter

Overview

The Field Data Submitter allows a party chief in the field to quickly, efficiently, and accurately send in data to office personnel. This application's main design philosophy is Usability in field conditions with simple layout, easy access, and redundancy. This means that this program strives to be straight forward, easy to use on field tablets, and be hard to fail or have an error. The application composes of an easy-to-use interface with built-in checks and securities. These features prevent accidental e-mail / data submissions as well as accidental touch protections.

Main Window

The main screen is where all the general actions take place. There are four things the user has to worry about; Job Number, Purpose, Files, Note. Ensuring that just this information is correctly inputted removes the possibility of messing up the "back-end" of the process. Further, if using the E-Mail method of data submission, this will automatically pre-format the e-mail to work with E-Mail Parser application. For more information on this, see Settings.

Job Number

The job number is the most important thing that you can enter. If this is wrong, the data you send in can be placed in the wrong project or not be fully processed at all.

Ensure you enter the correct job number before sending the data. Otherwise, you might look dumb.

You can enter the job number(s) one of two ways. The first is to just type the job number. The program will notify you if the number is not correctly inputted in the yy-mm-nnn format. If you have only one job number to submit, you can either leave this information in the box or click on "ADD" under the list to add the job number to the list. If you have multiple projects with one data submission, you MUST add each job number to the list using the "ADD" button under the job number list.
If you have a job number you wish to remove from the list, select the number and hit remove. This will automatically remove the number from the list.

Purpose

The purpose is just a reason for the data. In other words, why were you out there? You can either select a preset option from the dropdown list or type in your own value.

Files

The files you wish to send in. This can include your text file, AutoCAD® file, or anything else (including pictures). When you press "ADD", a dialog will open to select files (you can select multiple at once). To remove a file, select the file in the list and hit remove. Additionally, the initial directory can be changed via the Default Directory setting in the options menu.

Note

Any additional message you want to send about the data. If installed, the office personnel using AutoCAD® Civil 3D® Plugin will see the note when they import the points into CAD. Please keep it short and to the point.
